Bem vindo, Usuário

Esta aplicação irá de auxiliar na visualização, validação e gravação de seus extratos bancários. Para acessar é só navegar nos meus superior. Veja algumas das funcionalidades:

  • Importação de arquivos do tipo OFX
  • Manipulação durante e após a importação

Ferramentas utilizadas para esta aplicação:

  • Backend desenvolvida em Aspnet Core 3.1, com EF Core, padrão MVC (camadas)
  • Frontend desenvolvida em Angular 8.2.13, com Bootstrap 4.3
